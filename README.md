### Available Scripts

- Clone this repository
```bash
$ git clone git@github.com:alonsofl/slowppi-frontend.git
$ cd slowppi-frontend/
```

- Run the app
```bash
$ npm start # Runs the app in the development mode.
```

- Or build
```bash
$ npm run build # Builds the app for production to the `build` folder.
```
